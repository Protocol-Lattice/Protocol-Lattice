# Protocol Lattice website

A dependency-light Node.js marketing site for Protocol Lattice. The hero uses a native WebGL canvas to render an interactive three-dimensional lattice without a third-party 3D runtime.

## Run locally

```bash
npm start
```

Then open `http://127.0.0.1:4173`.

Set `PORT` or `HOST` when you need a different local bind address:

```bash
PORT=8080 npm start
```
